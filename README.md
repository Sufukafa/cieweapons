# qCold's CIE Weapon Database 🎯
**qCold's CIE Weapon Database** — це професійна, інтерактивна веб-база даних, створена для відображення детальних характеристик зброї та боєприпасів з гри Arma Reforger (з фокусом на проєкт/сервер CIE). Проєкт розроблений з акцентом на чистий та сучасний UI/UX, точність ігрових даних та надійну функціональність.

### ✨ Посилання: [Перейти до бази зброї (Натисни тут)](https://sufukafa.github.io/cieweapons/)

### ✨ Основні можливості:
* 📊 **Детальні таблиці характеристик:** Повна інформація про доступну зброю та боєприпаси.
* 🔄 **Динамічне сортування:** Зручне сортування даних за будь-яким параметром (калібр, шкода, швидкість тощо).
* 🎨 **Сучасний дизайн:** Професійний інтерфейс з використанням кастомних шрифтів та продуманої кольорової палітри.
